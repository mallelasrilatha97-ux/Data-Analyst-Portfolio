# SQL Project: Customer Churn Analysis

## Project Title
Customer Churn Analysis Using SQL

## Project Description
Analyzed customer data to identify churn patterns, retention rate, and customer behavior using SQL queries. The analysis helps businesses understand why customers leave and identify strategies to improve customer retention.

## Dataset Fields
- Customer ID
- Subscription Status
- Monthly Charges
- Contract Type
- Region
- Customer Tenure

## SQL Analysis Performed
- Total customers analysis
- Churn rate calculation
- Customer retention analysis
- Region-wise churn analysis
- Contract type analysis

## Tools Used
- SQL
- Excel
- Power BI
