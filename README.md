# Data-Analyst-Portfolio
Built an interactive Superstore Sales Dashboard using Microsoft Power BI to analyze sales, profit, and regional performance. Developed DAX measures for KPIs such as total sales, average sales, and profit ratio, enabling stakeholders to identify top-performing regions and products for data-driven decision making.

# Superstore Sales Dashboard

This project analyzes sales and profit using Power BI.

## Tools Used
- Power BI
- SQL
- Excel

## Key Insights
- Identified top performing regions
- Analyzed sales trends
- Compared profit by category
